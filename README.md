# Publishing Project

This repository contains the assets and workflows for a publishing project, including manuscript drafts, research materials, academic papers, and production assets.

## Structure

- `manuscript/`: Contains all manuscript-related files, including chapter drafts, research, and outlines.
- `academic/`: Stores working papers and literature reviews.
- `production/`: Holds EPUB build scripts, Pandoc configuration, and cover assets.
- `handoff/`: Contains materials for publication handoff, such as ISBN, metadata, and back cover copy.
- `skills/`: Houses agent revision workflows.
