== LCCN PREASSIGNMENT ==

Apply for a Preassigned Control Number (PCN) from the Library of Congress.
Required for library catalog metadata and LoC CIP data block (Cataloging in Publication).
Apply before book goes to press — LoC requires the book to be sent to them post-publication.

Application: https://lccn.loc.gov
Eligibility: US publishers with established mailing address and prior publication record (or intent).
Processing: 2–3 business days for preassignment.

== BISAC CLASSIFICATION ==

distributor catalog routing, and library acquisition category.

== INGRAM SETUP NOTES ==

Distribution channel: Ingram Lightning Source (preferred over IngramSpark for trade publishers)
SJ import hub routing: Confirm with Ingram account rep that San Jose hub is designated for this title's
  primary distribution run. Request confirmation in writing before print run authorization.

Ingram requires:
  - Print-ready interior PDF (PDF/X-1a or PDF/X-4, embedded fonts, no RGB elements)
  - Print-ready cover PDF (same spec, with bleed, barcode box, spine width verified)
  - Title setup form completed via Ingram portal
  - Wholesale discount declaration (standard trade: 55%)
  - Return policy declaration (returnable recommended for library channel)

Spine width calculation:
  Formula: Page count × paper caliper
  60lb cream offset: 0.002252 inches per page
  At 260 pages: 260 × 0.002252 = 0.5855 inches
  Round to nearest 0.001 inch: 0.586 inches
  Add to cover template spine box before submitting cover PDF.
  CONFIRM with Ingram spec sheet for the exact paper stock selected at print setup.
